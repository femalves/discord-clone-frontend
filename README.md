### :point_up: Frontend

```bash

# Clone this repository
$ git clone https://github.com/femalves/discord-clone-frontend

# From the terminal, go to folder
$ cd discord-clone

# Install dependencies
$ yarn add

# Run in development mode
$ yarn start

# Go to http://localhost:3000

```
